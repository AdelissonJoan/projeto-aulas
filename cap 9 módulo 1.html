HTML-CSS Módulo 1 Capítulo 9

Página desenvolvida para adicionar dicas de códigos e funções em HTML5 e CSS3.

Trabalhando com Listas:

Listas Ordenadas:

Para criar uma lista ordenada utilizamos a tag <ol> que significa ordered list, ela irá criar uma lista númerada que irá atualizar seus números automaticamente independente da ordem (adição de tarefas da lista).

Dentro de toda tag <ol> deve-se usar a tag <li> de list, segue o exemplo abaixo:

                    <ol>
                        <li>Acordar</li>
                        <li>Ligar para o João</li>
                        <li>Tomar café</li>
                        <li>Escovar os dentes</li>
                    </ol>

Obs. A partir do HTML5 a tag </li> passou a ser opcional, então se vc quiser pode utilizar apenas a tag <li>.

Dentro da tag <ol> podemos utilizar alguns parâmetros para personalizar a nossa lista, são eles:

  • <ol type="1"> (Irá criar uma lista ordenada com números).
  • <ol type="A"> (Irá criar uma lista ordenada com letras maiúsculas).
  • <ol type="a"> (Irá criar uma lista ordenada com letras minúsculas).
  • <ol type="I"> (Irá criar uma lista ordenada com números romanos em maiúsculo).
  • <ol type="i"> (Irá criar uma lista ordenada com números romanos em minúsculo).

Dica: O parâmetro de uma lista ordenada por padrão é 1, se vc quiser mudar isso pode usar a tag: <ol type="1" start="5"> para que lista inicie a númeração apartir do 5.

Listas não Ordenadas:

Para criar listas não ordenadas utilizamos a tag <ul> que significa unordered list, ela irá criar uma lista sem númeração ou ordem.

Dentro de toda tag <ul> deve-se usar a tag <li> de list, segue o exemplo abaixo:

                <ul>
                    <li>Pão.</li>
                    <li>Leite.</li>
                    <li>Tomate.</li>
                    <li>Alface.</li>
                    <li>Manteiga.</li>
                    <li>Arroz.</li>
                    <li>Feijão.</li>
                </ul>

E assim como na lista ordenada podemos utilizar parâmetros para personalizar nossa lista não ordenada, e são eles:

  • <ul type="disc"> (Irá criar uma lista não ordenada com uma bolinha preta no inicio).
  • <ul type="circle"> (Irá criar uma lista não ordenada com um pequeno círculo no inicio).
  • <ul type="square"> (Irá criar uma lista não ordenada com um pequeno quadrado preto no inicio).

Dica: O parâmetro de uma lista não ordenada por padrão é type="disc".

Obs. Geralmente esse tipo de lista não ordenada é bastante utilizado, principalmente na criação de menus.

Misturando Listas:

Podemos Misturar listas tanto ordenadas como não ordenadas da seguinte forma; criamos uma lista ordenada, e dentro dela uma lista não ordenada. Segue o Exemplo abaixo:

                <h2>Minhas Linguagens Favoritas</h2>
                    <ol>
                    <li>Antigas</li>
                    <ol type="a">
                    <li>Clipper</li>
                    <li>Visual</li>
                    <li>Fortran</li>
                    <li>Delphi</li>
                    </ol>

Após isso podemos determinar se essa lista sendo ela númerica ou alfabetíca irá continuar apartir da primeira, Exe:

                Minhas Linguagens Favoritas:
                    1.Antigas
                        a. Clipper
                        b. Visual Basic
                        c. Fortran
                        d. Delphi
                    2.Novas
                        e. PHP
                        f. Python
                        g. JavaScript
                        h. Kotlin

Nesse exemplo acima para a lista 2 continuar apartir da sub-lista 1 foi usado o parâmetro start="5". Obs o start só funciona através de números, então devemos contar quantos itens possuem na lista 1 para poder colocar seu número de continuidade na lista 2. No caso na lista 1 (4 itens) então a lista 2 inicia apartir do número 5.

Também podemos misturar uma <ol> com uma <ul> seguindo a mesma lógica do exemplo acima.

Dica: para modificarmos varias tags ao mesmo tempo; exe de <ol> para <ul> devemos: clicar com o curso do mouse no meio da tag 'ol' e segurar a tecla Alt depois apagar o 'o' e trocar pelo 'u'.

Lista de Definição:

Para criar uma lista de definição utilizamos as tags <dl> (definition list) <dt> (termo) e <dd> (description). Essas tags iram criar uma lista com termo e definições.

HTML-CSS Módulo 1 Capítulo 12

Página desenvolvida para adicionar dicas de códigos e funções em HTML5 e CSS3.

Estilo CSS Inline:

Vamos começar agora a adicionar estilos em seus arquivos HTML, para isso podemos utilizar 3 formas de estilo em CSS, a primeira forma é a de Estilo Inline e ela é basicamente o que? Você adicionar estilos CSS linha por linha dentro das tags do seu conteúdo em HTML. Segue um exemplo abaixo:

            <body style="Background-color: black;"> 
            (Irá mudar a cor de fundo de todo o contéudo da sua área body).

Com o estilo inline você consegue colocar em cada tag um estilo, e nesse estilo você configura certas coisas.

            <h1 style="color: blue;"> (Irá mudar a cor do seu título h1).

A maior desvantagem do estilo inline é que você vai ter que escrever o estilo dentro de cada tag que você queira mudar e se por acaso seu arquivo tiver um número muito grande de tags isso vai dar muito trabalho além de demorar bastante tempo.

            <body style="Background-color: black; font-family: Arial, Helvetica, sans-serif;"> 
            (Irá mudar a cor de fundo e a fonte de todo o conteúdo da sua área body).

            <p style="text-align: justify;"> (Irá alinhar o texto dentro do parágrafo).

Essa forma de estilo é ruim pois ela polúi e desorganiza muito o seu código HTML, por que você começou a colocar coisas de estilo CSS junto com o seu conteúdo. E o pior de tudo é que se você for fazer alguma alteração de cor (por exemplo) você vai ter que alterar quase todo o código. O ideal seria você separar o que é estilo do que é conteúdo. Mas esse método não é muito utilizado no dia-a-dia, esse é mais usado em configurações pontuais, uma ou outra configuração muito pequena.

            <body style="font-size: 20px"> 
            (Irá mudar o tamanho da fonte dentro do body para 20 pixels).

Então em resumo, o CSS Inline significa que o estilo é feito na mesma linha que o CSS.

Estilos CSS Internos:

Esse outro método de Estilo é chamado de CSS Interno (ou CSS Local) ele consiste em aplicar um estilo de forma separada, onde você vê nitidamente oque que é estilo e oque que é conteúdo. Então segue o passo a passo abaixo para entender como ele funciona:

Primeiro de tudo vamos copiar o nosso arquivo ex013 e criar o arquivo ex014 apartir dele, só que apagando os estilos inline de antes. Depois disso vamos começar a aplicar os estilos Internos (locais).

Na área de configuração <head> antes ou depois do título vamos criar a tag <style> dentro dela vamos adicionar o nome do seletor que queremos adicionar o estilo entre "{}" (chaves) e dentro dessas chaves vamos colocar os estilo que queremos, exemplo: cor de fundo, tamanho, fonte, etc.. Segue o exemplo abaixo que você vai entender melhor:

            <style>
                body{
                background-color: blue;
                font-family: Arial, Helvetica, sans-serif;
                font-size: 20px;
            }

Nesse exemplo acima, o "body" é o nosso seletor, isso que dizer que os estilos que vamos aplicar irão modificar apenas o conteúdo que estiver dentro dessa área. Exe: cor, fonte, tamanho, etc..

            h1{
                color: red;
                background-color: blue;
                }

Nesse exemplo acima ele irá modificar a cor da letra e cor de fundo de todos os h1 (diferente do estilo inline de antes).

Outros exemplos:

            h2{
                color: red;
                }

Nesse exemplo acima ele irá modificar a cor da letra de todos os h2.

            p{
                text-align: justify;
                }
              </style>

Nesse exemplo acima ele irá ajustar o texto de todos os paragráfos. E perceba que esses seletores estão todos dentro de uma única tag de estilo (<style>) deixando o seu código muito mais organizado, separando os estilos do seu conteúdo.

Vantagens dos estilos internos:

Mais organização no seu código diferenciando claramente o seu conteúdo dos estilo, melhor facilidade em modificar alguma coisa como: mudar cor, tamanho, etc..

Desvantagens dos estilos internos:

A área <style> acaba ocupando muito espaço no seu código, e se por acaso você quiser utilizar esses mesmos estilo em outras página do seu site você terá que copiar esses estilos e colar em todas as outras páginas.

Então esse método ele é útil quando você vai utilizar ele em apenas um documento, já em 20 ou 30 não compensa pois você teria que modificar também todos esses outros.

Estilos CSS Externos:

Estilos externos se constitui em utilizar os estilos CSS em um outro arquivo externo, separado do nosso conteúdo.

Usando o mesmo arquivo do ex014 vamos copia-lo e criar o nosso ex015, após isso vamos começar a trabalhar com nosso Estilos CSS Externos. E assim como fizemos no ex014 vamos apagar (ou recortar) o estilo utilizado antes e criarmos o novo, mas antes disso vamos criar um arquivo para uma página 02 e um <a href=""> para uma âncora, e depois disso vamos criar o nosso arquivo CSS externo, e pra criar ele é muito fácil, segue o passo a passo:

Na área de configuração <head> abaixo do title vamos criar um link:css:, assim que esse link é criado ele te dá uma sugestão de criar um arquivo em CSS (Usando o Ctrl+Clique), basta cria-lo e depois colar o "Estilo Interno" que você criou antes. Segue abaixo o exemplo do link:css: criado:

<link rel="stylesheet" href="style.css">

Obs: O arquivo CSS criado não tem compatibilidade com acentuação, então você precisará criar uma linha de regra UTF-8, dessa forma aqui: @charset "UTF-8";

Então basicamente esse método acima irá criar um arquivo para armazenar os seus estilo, e para usá-los basta apenas você carrega-los. Inclusive você pode utilizar mais de um link:css nos seus arquivos. E também ainda se você quiser pode misturar os estilos, usando CSS Interno, CSS Externo e CSS Inline no mesmo arquivo HTML. Exemplo:

Na área de configuração <head>:

            <link rel="stylesheet" href="style.css">
            <style>
                h1 {
                    text-decoration: underline;
                } 
            </style>

Veja que nesse método acima foi usado tanto o Estilo Externo quanto o Estilo Interno, (nesse exemplo de h1 o Estilo irá mostrar o título sublinhado na página).

Na área de conteúdo <body>:

            <p style="text-align: right;">
            <a href="pag02.html" target="_self">Ir para a página 2</a></p>

Nesse exemplo acima na área <body> do mesmo arquivo foi usado o Estilo Inline, (nesse exemplo de p o Estilo Inline irá mostrar o texto alinhado a direita).

Qual o melhor estilo a ser utlizado?

Estilo CSS Inline: evite utiliza-lo a todo custo, a menos que seja para uma configurção bem especifica em um caso bem especifico.

Estilo CSS Interno: só utilize em casos de estilos pequenos, se o seu estilo for crescendo e começar a atrapalhar o seu conteúdo HTML não utilize.

Estilo CSS Externo: use a vontade, sem moderação, e principalmente se o seu site tiver mais de uma página que for utilizar esse mesmo estilo.

De forma resumida, guarde isso na sua cabeça:

  • Estilo CSS Inline: procure evitar
  • Estilo CSS Interno: use para pequenas configurações
  • Estilo CSS Externo: use sempre que puder

Links e Âncoras em HTML:

Quando vc vai acessar um site e clica em algum link que te direciona para outro site isso é oque chamamos de HiperLinks ou Âncora. Todo HiperLink é construido atráves de uma âncora em HTML, e eles são muito importantes pois quando o mecanismo de buscar do seu navegador for procurar informações referentes ao seu site ele vai pesquisar por HiperLinks ligados a ele. ou seja, os HiperLinks ajudam a encontrar melhor o seu site.

Vamos supôr que vc acabou de criar o seu site, e vc quer "cadastrar" ele no google para que outras pessoas possam emcontrar e acessar ele, então; como podemos fazer isso? Atráves de HiperLinks, vc que tem um canal no youtube, um repositório no GitHub, ou até mesmo uma conta do instagram, basta criar um HiperLink uma Âncora e isso irá facilitar que o Google encontre melhor o seu site.

Dica: Para entender um pouco melhor como funciona essa busca, pesquise por: SEO (Searching Engine Optimization) que é Otimização para Mecanismo de Busca. Você vai organizar o seu HTML organizar o seu código justamente para os mecânismos de buscas como o Google te encontrar.

Links Externos:

Oque é um link externo? ele é um link que vai te enviar para outro site que não é o nosso. Agora vamos supôr que eu quero criar o seguinte Exemplo 01:

<p>Você pode acessar o meu repositório público no GitHub.</p>

E quando eu clicar em 'repositório público no GitHub' eu serei mandado para ele. Então para isso utilizamos a Tag <a> que por padrão já vem com o parâmetro 'href=' que significa Referência a HiperTexto que é pra onde o meu site irá. Então a Tag ficaria assim:

<p>Você pode acessar o meu <a href="hhtp://gustavoguanabara.github.io/">repositório público no GitHub</a></p>

Obs. nessa 'href=' vc deve colocar uma URL completa de pra onde vc quer mandar o seu site.

Só que ai temos um pequeno problema, quando a pessoa que estiver acessando o seu site clicar nesse link ele será enviado para ele e não irá mais voltar para o seu site (site anterior), então para fazer com que o Link Externo utilizado seja aberto em outra página e seu site continue aberto utilizamos os parâmetros target="_blank" (irá criar um alvo em uma janela em branco) e rel="external" (irá indicar que esse é um link Externo).

Basicamente o 'target' irá criar uma nova aba, fazendo assim com que o link externo não sobreponha o seu site. E o 'rel' irá indicar para o navegador que aquele é um link externo, de fora do seu site.

Exemplo 02: <p>Você pode acessar o meu <a href="https://youtube.com/cursoemvídeo/" target="_blank" rel="external">canal gratuito no YouTube</a></p>

Então basicamente no Exemplo 01 ele irá abrir o link externo e fechar o seu site, e no exemplo 02 irá abrir o link externo em uma nova aba e mânter o seu site aberto.

Então quando vc quiser redirecionar o usúario para outro site que não seja o seu usando um link externo é recomendado utilizar os parâmetros 'target' e 'rel', Mas se for para redirecionar pra outra parte do seu próprio site não é recomendado utilizar esse método.

Usando Links Internos:

Link interno, ele irá fazer a ligação de uma página com outras que estiverem dentro do nosso servidor, sendo elas relacionadas ao nosso site. e para criarmos um link interno nós precisamos de outras páginas, então vou ensinar duas formas de cria-las.

Link Interno Através de Arquivo:

Primeira forma: Com o VSCode aberto no seu index.html (sua página principal) vc irá até a opção 'criar um novo arquivo' e criará um pag002.html, depois irá criar o código base '!', o title, e o seu conteúdo (assim como vc fez no seu index.html). Após isso vc precisará criar um parâmetro para poder acessar a sua pag002, e para isso vc irá fazer o seguinte exemplo abaixo, na sua página 1 (index.html) você irá digitar:

<p>Esta é a primeira página do site. Se você quiser, pode acessar também a minha <a href="pag002.html">segunda página</a></p>.

Dica: dentro do href vc irá apertar Ctrl+Space e selecionar o arquivo 'pag002.html'. Nesse primeiro exemplo o parâmetro só irá funcionar se a pag002.html estiver na mesma pasta da página principal (index.html).

Agora vamos supôr que da página 002 vc queira criar um link para voltar pra página 001 (index.html), pra isso na página 002 utilizando o mesmo conceito do exemplo acima vc irá digitar:

            <p>A segunda página está aqui.</p>
            <p><a href="index.html">Voltar para a primeira página</a></p>.

Para facilitar com que o google saiba qual é a sua Próxima página ou a Anterior podemos utilizar os parâmetros: rel="next" (que significa próximo) e rel="prev" (que significa anterior).

            Exe 01: <p>Esta é a primeira página do site. Se você quiser
            pode acessar também a minha <a href="pag002.html" rel="next">segunda página</a></p>. 
            (Está indicando para o mecânismo que a página seguinte é a "pag002.html")
            
            Exe 02: <p>A segunda página está aqui.</p>
            <p><a href="index.html" rel="prev">Voltar para a primeira página</a></p>.
            (Está indicando para o mecânismo que a página anterior é a "index.html")

Essa próxima Tag mostra um parâmetro para por exemplo; uma propaganda que alguém pagou para vc mostrar no seu site e vc quer que ela seja mostrada mas o mecânismo de busca não siga além disso, apenas mostre ela e ponto.

            Exe 03: <p>Está página é um oferecimento de <a href="https://www.hostnet.com.br" 
            target="_blank" rel="nofollow">Hostnet</a></p>

Link Interno Através de Pasta:

Segunda forma: Com o VSCode aberto na sua página principal vc irá até a opção 'criar uma nova pasta' e dentro dela criará um novo arquivo pag003.html, depois irá criar o código base '!', o title, e o seu conteúdo (assim como no anterior). Após isso vc precisará criar um parâmetro para poder acessar a sua pag003, assim como vc fez na sua pag002, só que dessa vez vc terá que indicar dentro da URL especificamente em que pasta está o arquivo pag003.

            Exe: <p>Você também pode acessar a nossa <a href="noticias/pag003.html" 
            rel="next">página de notícias</a></p>

Dica: dentro do href vc irá apertar Ctrl+Space e selecionar a pasta noticias e depois o arquivo 'pag003.html'.

Após isso, ainda na sua pag003 vc irá criar o seguinte parâmetro abaixo para poder retornar da sua pag003 para a pag001 (index.html).

            Exe: <p>A sua terceira página está aqui.</p>
            <p><a href="../index.html" rel="prev">Voltar para a primeira página</a></p>.

(Como o arquivo da página 003 está em outra pasta diferente do arquivo da pag 001, você terá que usar '../' dentro do link de referência, para indicar que a pág003 deve retornar para um outro arquivo de outra pasta). Obs. './' significa na própria pasta, e '../' significa na pasta anterior.

Caso o arquivo esteja dentro de uma pasta vc tem que informar especificamente o caminho dele dentro da URL, para que o mecânismo de busca possa encontra-lo. Nesse caso de link internos para o próprio site; podemos também utilizar um parâmetro target="_self" para abrirmos as novas páginas na própria aba do nosso site (mas isso é opcional).

Parâmetros href e target:

1. hreflang="en" Irá indicar ao navegador qual o idioma principal do site para o qual aquele link está nos mandando.

            Exe: <a href="url" hreflang="pt-br"> (site em pt-br).

Por padrão, sempre que alguem clica em um hyperlink, o site de destino abre na mesma janela do site que continha esse link. Ou seja, o conteúdo anterior vai deixar de ser exibido para mostrar o novo conteúdo. Esse é um comportamento desejado apenas quando o visitante vai continuar a acessar o nosso site, apenas mudando de um documento para outro. Mas e quando um link leva o visitante para um outro site e ele provavelmente nunca mais voltará ao nosso?

para podemos controlar onde o site de destino vai abrir, podemos usar o atributo target, que suporta os seguintes valores:

  • 1. target="_blank" Vai abrir o link em uma nova janela em branco.
  • 2. target="_self" Vai abrir o link na janela ou frame atual (padrão).
  • 3. target="_top" Vai desfazer todos os frames e abrir o destino no navegador completo.
  • 4. target="_parent" Vai ser similar ao uso do _top Em uma referência a janela mãe.
  • 5. Nome-do-frame caso esteja usando frames, vai indicar o nome da janela a abrir.

Mas as mais utilizadas são _blank e _self.

Esse link é seu ou dos outros?

Existe um recurso bem interessante para links, que é indicar qual a natureza do destino dele usando o atributo "rel". Esse atributo aceita vários valores, como por exemplo:

  • 1. rel="next" Indica que o link é para a próxima parte do documento atual (ex: pag002).
  • 2. rel="prev" Indica que o link é para a parte anterior do documento atual (exe: pag001).
  • 3. rel="author" Indica que é um link para o site do autor do artigo atual (exe: index.html).
  • 4. rel="external" Indica que é um link para um site que não faz parte do site atual, um site externo (exe: YouTube.com)
  • 5. rel="nofollow" Indica que é um link para um site não endossado, como por exemplo um link pago.

Link Local (interno) e Link Externo:

1. Link local é aquele link que leva o visitante a outra página dentro do nosso próprio site, Nesses casos não é necessario indicar uma URL completa.

2. Link Externo é aquele link que leva o visitante para um outro site, e é necessario indicar uma URL completa e o caminho que leve a uma página em especifico.

Links para Download:

Agora vamos supôr que vc queira criar um link para download de um arquivo dentro do seu site, para isso vamos primeiro criar uma lista não ordenada e dentro dela criar 2 sub-listas com os arquivos de download e o seu caminho. Lembrando que esse método só irá funcionar se a pasta com os arquivos de download estiverem dentro da pasta do exercicío. Segue o exemplo abaixo:

            <ul>
            <li><a href="livros/meulivro.pdf" download="meulivro.pdf"
             type="application/pdf">Baixar o livro em PDF</a></li>

            <li><a href="livros/meulivro.zip" download="meulivro.zip"
             type="application/zip">Baixar o livro em ZIP</a></li>
            </ul>

Apartir da versão HTML5, as âncoras (hyperlinks) receberam atributos especiais para efetuar downloads de arquivos em PDF,ZIP e tec.. Para isso basta fazer o link diretamente para o caminho que se deseja efetuar o download e adicionar o atributo download="" Com o valor configurado para o nome do arquivo a ser baixado, e o atributo type="" para indicar ao navegador que tipo de arquivo será baixado.

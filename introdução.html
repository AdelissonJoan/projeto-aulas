HTML-CSS Módulo 1

Página desenvolvida para adicionar dicas de códigos e funções em HTML5 e CSS3.

Sobre o Curso, o que vamos aprender?

Evolução da internet, e como ela surgiu.

Diferença entre Domínio e Hospedagem.

Diferença entre Back-end, Front-end, Full-stack.

HTML = Conteúdo:

  • textos.
  • Imagens.
  • vídeos.
  • tabelas.

CSS = Design:

  • cores.
  • sombras.
  • tamanhos.
  • posicionamento.

JavaScript = Interações:

  • menus.
  • animações.
  • popups.
  • validações.

Módulos do Curso:

Módulo 1 primeiros passos HTML-CSS:

Conceitos básicos, preparação do ambiente, semântica da HTML5, textos, títulos, ligações, multimidia, estilos.

Módulo 2 deixando as coisas mais bonitas:

Fundamentos do design, psicologia das cores, tipografia, elementos CSS, modelo das caixas, wireframe, responsividade.

Módulo 3 colocando um protótio no ar:

Versionamento de software, hospedagem de sites estáticos, tabelas.

Módulo 4 aprofundando os conhecimentos:

Quadros em linhas, formulários, media queries, mobile first.

Módulo 5 novas tecnologias:

Flexbox, grid layout, projeto final.

4. Estude e tente estabelecer rotinas, disponibilize um tempo do seu dia.
